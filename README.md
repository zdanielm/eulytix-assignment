# eulytix-assignment
 Eulytix Assignment

## Setup (using pip):

### 1.: Create Python venv

```bash
python -m venv .venv  # or python3 -m venv venv
```

### 2.: Activate the venv

#### Windows
```bash
.venv\Scripts\activate
```
#### Linux & Unix

```bash
source .venv/bin/activate
```

### 3.: Install required packages with pip (Streamlit, OpenAI, PyPDF2)
```bash
pip install -r requirements.txt
```
